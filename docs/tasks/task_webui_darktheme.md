# Интерфейсные улучшения для веб-интерфейса

У меня есть код для lib\ETLTest\etl_wifi_setup*, который служит для настройки сети, в которой будет работать устройства на базе ESP8266/ESP32. 

Необходимо выполнить добавить несколько визуальных улучшений.

## Требования при работке

- Разрабатывать с учетом правила docs\rules\qwen_make_no_mistakes.md
- Этот код после отладки будет помещен в бибилиотеку ETL (https://github.com/JimorMarlow/ETL). Поэтому может использовать все неоходимое из нее и использовать общий стиль оформления.
- Код должен работать на ESP8266 и ESP32. Участка кода, специфические для каждого контроллера разделять с помощью нужных дефайн. Пример:
```cpp
// Для включения нужной wi-fi библиотеки
#pragma once
#ifdef ESP8266
  #include <ESP8266WiFi.h>
#elif ESP32
  #include <WiFi.h>
#else
  #pragma message("ERROR: no Wi-Fi lib specified")
#endif
```
- После исправлений необходимо протестировать компиляцию всех конфигураций из platformio.ini
Нужно выполнить:
C:\Users\amber\.platformio\penv\Scripts\platformio.exe run -e <имя_конфигурации> -d c:\Projects\Arduino\ETLTest
- По запросу сделать текст для коммита, но самому в git не выкладывать
- Для обновления контеста можешь посмотреть историю коммитов
- Все изменения записывай для себя в docs\tasks\task_webui_darktheme_hystory.md, чтобы потом можно было продолжить

## Расширить настройки структурой view
 - В настройки etl::wifi::server_config_t добавить поля для следующих возможностей. Добавить все поля из следующих пуктов, сделать сохранение настроек и трассировку. Отделить комментарием, как "Настройки интерфейса"

### Языковые настройки

Сейчас в интерфейса настройки wi-fi есть две кнопки [EN] [RU] для переключения между языками. Нужно сделать более универсальный механизм, предусмотреть, что в дальшейнем могут появиться другие языки.

- Каждый язык определеяется строкой константой из двух симоволов, придерживаться стандарта двухбуквенных кодов ISO 639-1. Код должен быть оптимизирован для ручного или автоматизированного добавления нового языка и все изменения должны быть сделаны только в нем
- в lib\ETLTest\etl_wifi_setup добавить список доступных языков. Вместо двух кнопок [EN] [RU] сделать одну. По умолчанию язык анлийский, на кнопке должен отобраться язык, на который мы переключимся, если нажмем. Языки переключаются по кругу. 
- В настройки etl::wifi::server_config_t добавить поле language для сохранения языка, по умолчанию EN.

### Светлая/темная тема

Добавить переключение темы со светлой на темную.
- В настройки etl::wifi::server_config_t добавить поле bool dark_theme, по умолчанию false (светлая тема)

### Увеличенный шрифт интерфейса

Добавить переключение темы со светлой на темную.
- В настройки etl::wifi::server_config_t добавить поле bool ui_scale, по умолчанию false (нормальный размер шрифта)
Если true - то увеличенный для людей с ослабленным зрением, или для мелких экранов
- Убрать кнопку [AA] из верхней части интерфейса, оставить только переключение языков

### Bold шрифт для ключевых значений

Режим для выделения ключевых цифровых значений без изменения размеров шрифта.
- В настройки etl::wifi::server_config_t добавить поле bool use_bold_values, по умолчанию false (нормальный вес шрифта)

## Структура страницы

```
┌─────────────────────────────────────────────────────┐
│ Settings / Настройки                        [RU]    │ ← header (data-i18n="title")
├─────────────────────────────────────────────────────┤
│ [Device Icon]  Device Name                           │
│                Description                           │ ← device-info-container
├─────────────────────────────────────────────────────┤
│ Настройки интерфейса                                │ ← ui-settings-container
│ ┌─────────────────────────────────────────────────┐ │
│ │ Тёмная тема                              [toggle]│ │
│ │ Увеличенный шрифт                        [toggle]│ │
│ │ Ключевые значения (Bold)                 [toggle]│ │
│ └─────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────┤
│ WiFi Setup / Настройки WiFi                         │ ← section-title (data-i18n="main_title")
├─────────────────────────────────────────────────────┤
│ [●] Статус подключения              [Refresh]       │ ← status-section
│     IP адрес                                        │
├─────────────────────────────────────────────────────┤
│ Select Network / Выберите сеть                      │ ← section-title
│ ┌─────────────────────────────────────────────────┐ │
│ │ 📶 Network Name           🔒                    │ │ ← networks-list
│ │     Excellent • WPA2                            │ │
│ └─────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────┤
│ Access Point Settings / Настройки точки доступа     │ ← ap-settings-section
│ ┌─────────────────────────────────────────────────┐ │
│ │ AP SSID:     [ESP_Device_AP]                    │ │
│ │ AP Password: [••••••••••]  [Show]               │ │
│ └─────────────────────────────────────────────────┘ │
│ [Apply AP Settings]                                 │
├─────────────────────────────────────────────────────┤
│ [Save & Reboot]                                     │
│ [Factory Reset]                                     │
└─────────────────────────────────────────────────────┘
```

**Элементы управления:**
- **header** — заголовок страницы с кнопкой переключения языка
- **device-info-container** — информация об устройстве (иконка, название, описание)
- **ui-settings-container** — настройки интерфейса с iOS-переключателями
- **section-title** — заголовок раздела (стиль как у "Select Network")
- **status-section** — статус подключения с индикатором и кнопкой обновления
- **networks-list** — список доступных WiFi сетей
- **ap-settings-section** — настройки точки доступа
- **btn** — кнопки действий (Save & Reboot, Factory Reset)

**Особенности:**
- При включении опции "Увеличенный шрифт" (large_font) все текстовые элементы увеличиваются на 20%
- Переключение языков работает по кругу: EN → RU → EN
- Настройки интерфейса сохраняются в постоянной памяти через API /api/ui_settings
